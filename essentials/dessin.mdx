# Dessins des Compositions et Alignements

Ce document complète `group-style-system.md` avec des schémas ASCII pour visualiser rapidement les variantes de layout et les options d'alignement exposées dans le builder.  
Chaque bloc montre la structure générale ; les lettres représentent l'ordre des médias.

---

## 1. `single`

Layout disponible : `horizontal`

```
┌───────────────┐
│      A        │
└───────────────┘
```

Alignement (vertical) : Top / Middle / Bottom

### Légende Alignement

- **Top / Middle / Bottom** : alignement vertical lorsque le conteneur principal est en `flex-row` ou `grid`.
- **Left / Center / Right** : alignement horizontal lorsque le conteneur principal est en `flex-col`.
- Les mosaïques utilisent un mix grid/flex, mais héritent des mêmes conventions (le bloc principal reste centré ; les sous-grilles garantissent le contact bord à bord avec l'image principale).
